OrgBook API

Are you developing an application geared towards registered organizations in BC?

OrgBook BC has a REST API (the OrgBook API), allowing developers to integrate publcily available, verifiable data about registered BC organizations into third-party applications.

A GitHub repository has been created as a documentation source for the OrgBook API, along with guides for the most common use cases of the API, including those that OrgBook BC implements.

Some of these use-cases include:

  • Autocomplete-enabled organization name search
  • Organization data retrieval, in the form of verifiable credentials (including registration number, business number, entity type, entity status, etc.)
  • Credential issuer search
  • Credential type search
  • Credential verification
